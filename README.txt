Le projet est un explorateur de fichier.

Le logiciel ne requiert aucune installation particulière.
1 - Unzip le fichier .zip dans un répertoire de votre choix.
2 - Ouvrir un terminal python.
3 - Aller jusqu'au répertoire où le fichier .zip a été unzip.
4 - Écrire dans le terminal python : python fileBrowser.py

Une fois le logiciel lancé, le logiciel se comporte comme tout autre explorateur de 
fichier comme celui de Windows ou de Linux.

Le programme contient les fonctions de bases qui consistent à un explorateur de fichier:
1 - Effacer des fichiers et / ou des répertoires.
2 - Copier des fichiers et / ou des répertoires.
3 - Renommer des fichiers et / ou des répertoires.
4 - Couper des fichiers et / ou des répertoires.

Les boutons en bas à droite de la fenêtre sont tous explicites.

***MENTIONS SPÉCIALES***
Pour l'utilisation de l'action cut, choisir le fichier à cut. Ensuite, sélectionner le 
fichier, appuyez sur le bouton "Cut File". Aller dans le répertoire où on veut placer le 
fichier. Appuyez sur Paste File. L'action se fait en arrière-plan; le fichier sera effacé 
sa première position et sera transféré dans le nouveau répertoire.

On ne peut pas changer de root directory. L'explorateur de fichier restera toujours dans 
le root directory de l'ordinateur.
